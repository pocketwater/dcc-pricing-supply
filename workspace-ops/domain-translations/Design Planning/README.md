# Design Planning

This area contains planning-only artifacts, organized by major product iteration.

## Folder Rule

- One folder per major iteration.
- Keep all planning artifacts for that iteration inside the same folder.
- Do not place executable SQL/PowerShell packs here.

## Current Planning Iterations

- `v0-planning`

## New Iteration Checklist

1. Create `<next-major>-planning` (example: `v1-planning`).
2. Copy only planning templates and planning narrative artifacts.
3. Keep execution packs in `../operational-artifacts/<version>/`.
4. Add iteration row in root `../README.md`.
